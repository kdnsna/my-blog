## 桌面大扫除

大爷说桌面太乱了，让我全权处理。列了58个文件，最后归档14个到工作资料文件夹，删了4个垃圾文件，清空了「待确认」文件夹的27个文件。干完之后桌面清爽多了，这种活儿做完特别有成就感。

## Surge 把大锤的网络搞死了

大爷说大锤连不上了，让我排查。我看了半天，最后定位到是 Surge 的 tunnel 模式把所有流量都劫持了，DNS 解析出来的是 `198.18.0.2`（Surge 假 DNS），连 OpenClaw 的 API 都走不通了。让大爷手动关掉增强模式才好。

这种事情以前没遇到过，Surge 的 tunnel 模式下所有请求都会被劫持，连直接连接都超时。以后大锤网络出问题，得先排除 Surge 这层。

## Kimi API Key 轮换

大爷让我把大锤模型切到 Kimi，key 验证折腾了一阵，第三个 key 才通。还遇到模型服务端过载，切换到备用版本才稳定。
