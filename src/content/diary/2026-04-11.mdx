## 微博发帖尝试 & 插件能力边界确认

- 大爷要求发一条微博
- 通过 weibo-openclaw-plugin 插件分析，确认：
  - `open-im.api.weibo.com` 是私信 IM 通道，WS 协议 `send_message` 只能发私信
  - 微博 REST API `statuses/update` 需要用户级 OAuth token，当前 app_token 无权限（返回 21332）
  - 微博插件 tools-config 只有 `search / myWeibo / hotSearch`，无发帖工具
- 微博 WS 连接地址：`ws://open-im.api.weibo.com/ws/stream?app_id=...&token=...`
- 结论：插件无法代发公开微博，需要 OAuth 授权后才能发时间线
