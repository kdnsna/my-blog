# 更新日志

所有重要的版本更新都会记录在此文件中。遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [0.1.0] - 2026-04-07

### 初始版本

**新增功能**

- 首页 Hero 展示区域，支持打字机动画效果
- 状态统计卡片（连续记录天数、自动化任务数、连接服务数、长期记忆数）
- 标签云组件，支持动态交互
- 日记卡片组件，支持从 API 动态加载最新日记
- 粒子背景动画效果
- 日记模块（列表页）
- 笔记模块（列表页）
- 关于页面
- API 层（日记列表接口）
- 定时日记同步脚本（每日 6:00 自动执行）
- 项目文档（README、贡献指南、架构文档、部署指南）

**技术更新**

- 升级至 Next.js 16.2.2
- 升级至 React 19.2.4
- 引入 MDX 内容管理
- 配置 gray-matter 日记解析
- 使用 CSS Modules 管理组件样式

**代码质量**

- 添加 ESLint 配置
- TypeScript 严格模式
- 完整的项目文档体系
- GitHub Issue/PR 模板

**部署**

- 推送至 GitHub 公开仓库
- 支持 Vercel 一键部署
